**支持的命令**


## 📖 使用指南

### 本地运行

```bash
cd git-tutorial-website

# 安装依赖
npm install

# 开发模式
npm run dev
# 访问 http://localhost:3000

# 生产构建
npm run build
npm start
```

### 添加新教程

1. 在 `content/beginner|advanced|projects/` 创建 `.mdx` 文件
2. 添加 frontmatter：

```yaml
---
title: "教程标题"
description: "教程描述"
category: "beginner"
level: "入门"
duration: "15 分钟"
tags: ["标签1", "标签2"]
date: "2024-01-01"
---
```

3. 编写 Markdown 内容
4. 自动集成到网站

### 添加新练习题

编辑 `lib/exercises.ts`，添加新的 Exercise 对象。
