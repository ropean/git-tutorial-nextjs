# 快速开始

欢迎来到 Git 快速开始教程！本章节将帮助你在 30 分钟内快速上手 Git。

## 学习目标

完成本章节后，你将能够：

- ✅ 理解 Git 是什么以及为什么要使用它
- ✅ 在你的电脑上成功安装和配置 Git
- ✅ 创建你的第一个 Git 仓库
- ✅ 掌握基本的 Git 工作流程

## 章节内容

本章节包含以下内容，建议按顺序学习：

1. [Git 简介](introduction/) - 了解 Git 的基本概念
2. [安装配置](installation/) - 在不同操作系统上安装 Git
3. [第一个仓库](first-repository/) - 创建并管理你的第一个项目
4. [基本工作流程](basic-workflow/) - 学习日常使用的基本命令

## 预计学习时间

⏱️ **约 30 分钟**

## 前置要求

- 无需任何 Git 或版本控制经验
- 基本的计算机操作能力
- 了解命令行基础知识会有帮助（但不是必需的）

让我们开始吧！
